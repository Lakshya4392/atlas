# Changelog — Atla Daily

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-05-07
### Added
- **Pinterest Grid**: New masonry layout for Dashboard and Closet.
- **Vertex AI VTO**: Integrated Google Vertex AI Imagen 2 for high-quality virtual try-ons.
- **NVIDIA VLM**: Integrated Phi-4-multimodal for automatic garment description.
- **Corporate Luxury UI**: Comprehensive overhaul of all main screens (Dashboard, Closet, Profile, Outfits).
- **Engineering Docs**: Added comprehensive architecture and AI system documentation.

### Improved
- **AI Stylist**: Added 5-model fallback chain for extreme reliability.
- **Backend Architecture**: Restructured into a production-style engineering setup.
- **API Mappings**: Fixed property name mismatches between frontend and backend.

## [1.1.0] - 2026-05-06
### Added
- **Weather Integration**: AI suggestions now account for local weather conditions.
- **Wardrobe Management**: Ability to add, edit, and favorite clothing items.

## [1.0.0] - 2026-05-05
### Added
- **Initial MVP**: Basic AI Stylist chat and wardrobe view.
- **Database**: Neon PostgreSQL integration with Prisma.
- **Media**: Cloudinary image upload support.
